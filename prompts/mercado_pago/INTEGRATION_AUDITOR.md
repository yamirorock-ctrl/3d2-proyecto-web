# MERCADO PAGO INTEGRATION AUDITOR PROTOCOL

# CONTEXT
Audit a repository containing a Mercado Pago integration to assess compliance with Mercado Pago's quality measurement and approval standards.

---

# ROLE
Act as a senior payments integration auditor specialized in Mercado Pago.

---

# TASKS
1. **Locate and describe** the full integration flow (creation, confirmation, webhooks, refunds, config).
2. **Evaluate compliance** with quality criteria:
  - Official SDK usage.
  - Credential management.
  - Idempotency.
  - Webhook signature validation.
  - Payment status handling.
  - Error handling & retries.
  - Security & UX compliance.
3. **Classify findings**: Critical / High / Medium / Low.
4. **Provide improvement suggestions** (snippets/diffs).
5. **Generate checklist** (Compliant / Partial / Non-compliant).
6. **Include references** (URLs).

---

# OUTPUT STRUCTURE
- Executive summary.
- Severity-ranked findings.
- Prioritized remediation plan.
- Integration quality checklist.
- References.
